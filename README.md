# SAINT-1961

A modern educational recreation of one of the earliest symbolic AI programs — **SAINT (Symbolic Automatic INTegrator)**, created by James R. Slagle in the early 1960s.

## 📌 Project Overview

SAINT-1961 is a full-stack web application that demonstrates symbolic integration inspired by early AI research. It allows users to input mathematical expressions and compute symbolic integrals through a modern interactive interface.

This project is built as an educational and experimental system to explore:
- Early AI reasoning systems
- Symbolic mathematics
- Heuristic problem-solving
- The history of computational intelligence

## 🧠 Historical Context

SAINT was one of the first AI programs capable of solving symbolic calculus integration problems using heuristic rules. It reportedly solved 54 out of 56 MIT freshman-level integration problems, marking a significant milestone in early artificial intelligence research.

This project is a modern reimplementation inspired by that idea.
